O elemento blockquote é usado para indicar que o texto dentro dele é uma citação retirada de outra fonte. Normalmente, os navegadores exibem o texto dentro de um blockquote com recuo, para distingui-lo do restante do conteúdo. É comum ver blockquote sendo usado em artigos, posts de blog, etc., para destacar citações de outros autores.

Exemplo:

Como dizia Sócrates:

Só sei que nada sei


O elemento abbr é usado para definir uma abreviação ou acrônimo em uma página da web. Você pode usar o atributo title para fornecer uma explicação ou descrição do que a abreviação significa quando o usuário passa o mouse sobre ela. Isso é útil para tornar o conteúdo mais acessível.

Exemplo:
ONU

O elemento q é usado para indicar uma citação curta diretamente no texto. Assim como blockquote, q também é usado para citações, mas é mais adequado para citações curtas, como uma frase ou parte de uma frase.

Exemplo:

como dizia Sócrates só sei que nada sei


O elemento pre é usado para texto pré-formatado. Isso significa que o texto dentro de pre é exibido exatamente como está no código HTML, mantendo espaços em branco, quebras de linha e outras formatações. É útil para exibir blocos de código ou texto onde a formatação é crucial.

Exemplo: 
Opa 
                    Sabia 
                            que
             Sócrates 

                                 disse:

                        Só 
                  sei 
                         que                    sei
                                    nada
